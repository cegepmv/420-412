Cette section a pour objectif de transformer votre manière de concevoir une application React en créant une connexion fluide et robuste entre votre interface et votre API NestJS.

Nous allons partir des fondements acquis dans le cours **420-211 - Applications Web** pour pousser les concepts un peu plus loin. 

Pour mettre cela en pratique, nous allons travailler sur une application concrète. Nous utiliserons un projet frontend de départ contenant des données "en dur"(*mock data*) ainsi qu'un backend déjà fonctionnel :

+ Frontend de départ : [vanlife-frontend](https://github.com/gbenachour/vanlife-frontend)
+ Backend (NestJS) : [vanlife-api](https://github.com/gbenachour/vanlife-api)

{{%notice style="info" title="À Noter"%}}
Avec vos connaissances actuelles de React, vous avez déjà tous les outils nécessaires pour mener à bien votre projet. Ce chapitre est **optionnel** : il s'adresse à ceux qui souhaitent monter en compétence et découvrir des outils plus avancés comme les *Loaders*, les *Actions* et la gestion de données asynchrones moderne.
{{%/notice%}}

##### Comment suivre ce cours ?
Vous avez deux options selon votre préférence :
+ **Méthode pratique :** Cloner le dépôt du backend localement et faire un fork du frontend pour reproduire les étapes en même temps que votre enseignant.
+ **Méthode intégration :** Écouter les concepts et les implémenter directement sur votre propre projet de session.
